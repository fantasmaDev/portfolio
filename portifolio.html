Cronômetro Simples
Programa básico, sem interface gráfica (roda em terminal), cujo objetivo é registrar o tempo decorrido para realização de uma determinada atividade, de acordo com controle do usuário. Todos os intervalos cronometrados em um mesmo dia são salvos em um arquivo .JSON, de modo que é possível acessar o tempo total marcado naquela data específica.

O projeto foi implementado utilizando conceitos de orientação a objetos para definir as classes de 'tempo' e utilização de bibliotecas específicas do Python, como 'json' e 'datetime'.

Conversor de Formatos
Assim como o cronômetro, o programa em questão não possui interface gráfica. Foi criado com o objeto principal de converter extensões de diversos arquivos de uma única vez, sem necessiade de renomá-los manualmente. É possível assim, por exemplo, converter várias imagens de formato '.webp' para '.jpg'.

O projeto foi implementado utilizando diversas funções para manipulação de arquivos presentes nas blibliotecas 'os' e 'system' do Python.
